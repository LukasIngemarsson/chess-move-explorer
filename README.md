# Chess Move Explorer

Explore your opening tendencies by position across your game history.

Fetch your games from **Lichess** or **Chess.com**, then navigate the board to see which moves you and your opponents have played from any position — with frequency and win/draw/loss rates, filterable by time control.

## Stack

- [SvelteKit](https://svelte.dev) + TypeScript
- [chess.js](https://github.com/jhlywa/chess.js) — move parsing and FEN generation
- [Chessground](https://github.com/lichess-org/chessground) — board rendering
- [Tailwind CSS v4](https://tailwindcss.com) + [DaisyUI](https://daisyui.com)
